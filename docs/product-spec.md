# Snap Recorder MVP 产品规格

## 目标

用最短路径完成高质量 Mac 录屏，不进入编辑器，不要求用户设置格式或选择保存目录。

主流程：

未录人声：

`首次授权 → 选择来源 → 3 秒倒计时 → 录制 / 暂停 → 结束 → 自动保存到下载`

录制人声：

`开启人声并授权 → 录制 → 结束 → 勾选一种或两种导出版本 → 保存到下载`

## 两种来源

### 浏览器窗口

- 支持 Safari、Chrome、Edge、Arc、Firefox、Brave、Orion、Opera。
- 只抓选中窗口，其他应用和其他浏览器窗口不进入成片。
- 输出跟随开始录制时的浏览器窗口比例，不拉伸、不裁切；优先保留 Retina 原始像素，最高约 3840×2160。
- 当前桌面壁纸作为背景；读取失败时使用内置渐变。
- 壁纸轻微模糊与压暗；浏览器窗口约占画布 95% 宽高，等比居中，加圆角和柔和阴影。

### 整个屏幕

- 录制当前主显示器。
- 保持显示器原始比例，不拉伸、不裁切。
- 优先保留显示器原始像素，最大输出约 3840×2160。
- Snap Recorder 的窗口、倒计时和控制胶囊不会进入视频。

## 状态与反馈

- 未授权：只显示授权说明和唯一主按钮。
- 空窗口列表：明确提示先打开浏览器，并提供刷新。
- 倒计时：3、2、1，每个数字 1 秒；数字消失后等待约 120 毫秒再启动采集。
- 录制中：顶部胶囊显示红点、真实录制时长、暂停和结束。
- 暂停：红点变橙色，显示“已暂停”；继续时不再次倒计时。
- 处理中：显示“正在整理录制…”或“正在生成高清视频…”，不显示虚假百分比。
- 未录人声：停止后直接保存，不增加步骤。
- 已录人声：停止后显示“完整视频”和“视频和人声分轨”两张可多选卡，默认勾选完整视频。
- 完成：显示一个、两个或三个文件名、“在访达中显示”和“再录一个”。
- 失败：保留原因，返回后可以直接重试。

## 输出

- MP4 / H.264 / SDR / Rec.709。
- 默认 60 fps 采集，实时硬件编码。
- H.264 质量预算按像素数提升，约 24–68 Mbps；静态画面的实际平均码率可能更低。
- 电脑声音为 AAC 48 kHz 双声道，192 kbps。
- 人声默认关闭；开启后使用系统默认麦克风、AAC-LC 48 kHz 单声道、192 kbps。
- 合并导出为一个 MP4，音频混合为 AAC 48 kHz 双声道、256 kbps；视频轨不重新编码。
- 分开导出为 MP4（含电脑声音，如开启）与 M4A 人声，不打包；两个文件时长误差不超过 40 毫秒。
- 完整视频文件名：`Snap 录屏 YYYY-MM-DD HH.mm.ss.mp4`。
- 分轨视频文件名：`Snap 视频 YYYY-MM-DD HH.mm.ss.mp4`。
- 分轨人声文件名：`Snap 人声 YYYY-MM-DD HH.mm.ss.m4a`。
- 保存位置：`~/Downloads`。

## 不做

- 剪辑、变焦、光标轨迹美化、字幕、摄像头、云分享。
- 导出格式、分辨率、码率和文件夹选择。
- 麦克风设备选择、多显示器选择、区域录制。

## 首版验收

- 首次启动能解释权限用途，授权后可进入来源选择。
- 倒计时与工具界面均不出现在成片中。
- 浏览器成片不出现其他应用、Dock、菜单栏、桌面图标或通知。
- 暂停后继续录制，暂停空档从最终文件中移除。
- 结束后无需选择路径即可在下载目录获得 QuickTime 可播放的 MP4。
- 开启人声后，合并文件同时包含电脑声音（如开启）与人声，视频画质不下降。
- 分开导出正好得到 MP4 与 M4A 两个文件，时长对齐且不生成 ZIP。
- 同时勾选两种版本时，一次得到完整 MP4、分轨 MP4 和 M4A 共 3 个文件；任一生成步骤失败都不会删除原始素材。
- 来源关闭或编码失败时给出明确反馈，不静默丢失。
